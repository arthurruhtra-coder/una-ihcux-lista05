# una-ihcux-lista05
Resumo das Heuristicas utilizadas:
*. Status (Visibilidade do estado do sistema)

O sistema deve sempre manter o usuário informado sobre o que está acontecendo, com respostas rápidas e claras (ex: mensagens de carregamento, confirmação de ações, progresso).

*. Prevenção de Erros

O sistema deve evitar que erros aconteçam, guiando o usuário corretamente (ex: validação de campos, avisos antes de ações perigosas, opções seguras por padrão).

*. Estética (Design minimalista e agradável)

A interface deve ser simples, organizada e visualmente agradável, sem informações desnecessárias que possam confundir ou distrair o usuário.
